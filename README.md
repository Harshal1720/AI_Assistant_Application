An AI-powered career mentor web application built using Streamlit and Google Gemini API.

## Features
- Chat-based AI mentor
- FAQ buttons
- Streamlit UI
- Gemini model integration

## Run Locally

1. Install libraries
pip install -r requirements.txt

2. Create .env file
GOOGLE_API_KEY=your_api_key_here

3. Run app
streamlit run chat.py
